# О проекте
Наша работа - студенческий проект для компании *ООО "ИНПАД"*, выполнется в рамках Проектного Практикума.\
Данный репозиторий содержит в себе **3 отдельных микро-проекта**:
- Cервер с базой данных (ASP.NET Core + EF Core)
- Админ-панель (React-клиент)
- Плагин для программы Revit (C#)
# Cервер
**Стек: C# 8.0, ASP.NET Core 8.0, Entity Framework Core 8.0, библиотека Telegram.Bot 22**\
Сервер выполнен в монолитной многомодульной архитектуре с помощью ASP.NET Web API, на нём обрабатываются запросы:
- Telegram-бота
- Аутентификация, управление Базой Знаний админ-панели
- Плагина Revit

**Сервер разделен на 3 слоя:**
1. Controllers (для связи с другими сервисами Revit, Telegram, админ-панелью)
2. Data (Модели сущностей и конфигурация Базы Данных)
3. Services (Бизнес-логика проекта)\
*также есть вспомогательные слои, такие как:*
- **Migrations** для БД,
- **Propreties** c описанием зависимостей по технологии DI
- **DTOs** с моделями Data Transfer Objects для общения с админ-панелью\
