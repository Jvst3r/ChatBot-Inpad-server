# О проекте
Наша работа - студенческий проект для компании *ООО "ИНПАД"*, выполнется в рамках Проектного Практикума.\
Данный репозиторий содержит в себе **3 отдельных микро-проекта**:
- Cервер с базой данных (ASP.NET Core + EF Core)
- Админ-панель (React-клиент)
- Плагин для программы Revit (C#)
# Cервер
Сервер выполнен в монолитной многомодульной архитектуре, на нём обрабатываются запросы:
- Telegram-бота
- Аутентификация, управление Базой Знаний админ-панели
- Плагин Revit\

**Проект включает в себя папки:**
1. Controllers (для связи с другими сервисами Revit, Telegram)
2. Data (Модели сущностей и конфигурация Базы Данных)
3. Services (Бизнес-логика проекта)
